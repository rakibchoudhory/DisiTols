# DigiTools - Digital Product Store

**DigiTools** is a simple website where users can find and buy digital tools like AI writers and resume builders. I built this project to learn how to manage data and show it on a screen using **React**.

## Technology I Used
* **React.js**: A library to build the user interface.
* **Tailwind CSS & DaisyUI**: Tools to make the website look pretty and clean.
* **React-Toastify**: A package to show "success" or "error" messages to the user.
* **JSON**: Used to save and load all the product information.

---

## Features of this Website:

### 1. Smart Shopping Cart
When you click the "Buy Now" button, the product goes into your cart. The total number of items shows up in the top menu (Navbar) immediately so you always know what you are buying.

### 2. View Switching (Toggle)
I made two buttons in the center of the page. One button shows the **Product List**, and the other shows your **Cart**. This keeps the website clean and easy to use.

### 3. Real-Time Alerts
I used **React-Toastify** to help the user. If you add a product, remove one, or click "Checkout," a small message pops up to tell you what happened.
